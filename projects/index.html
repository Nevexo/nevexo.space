My Projects

Here you'll find a list of my active, retired and in development projects.

tmw.media API Active

One of my biggest projects to-date has been the tmw.media API, originally built for the GGRadio.net Internet Radio
this API provides real-time music information, including titles, artists, artwork and play time.
The API has had 5 revisions, three of which I have written. The API is now written in Node.JS and uses
Docker for fast deployment and scaling.
Being modular, this API now services other projects from tmw.media, including Virtual Market.

Mixer Retweet / Retweet X Active

Mixer Retweet version 3.0, code-named 'RetweetX' is a Twitter auto-retweet bot originally built for a Twitter account I co-own, Mixer Retweet.
This bot is written in TypeScript, features a web interface and is highly modular.

HeroPro In Development

HeroPro (name change pending) is a digital signage micro-service platform, written in TypeScript, C# and Python.
The original plan for HeroPro was a digital signage system that made use of Microsoft Windows' Screen saver system.
The project will expand on this to support many different viewing platforms, and use distributed micro-services to reduce network load.

George Greenaway Show Management System Retired

During the George Greenaway 24 and 36 hour radio marathons, I wrote a show Management system, collecting SMS, Facebook, Twitter and donation messages,
and placed them into a web interface, it also had a communication system for show crew, a note taking system and used WebSockets to deliver
new messages in near real-time.

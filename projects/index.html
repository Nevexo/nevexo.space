My Projects

Here you'll find a list of my active, retired and in development projects.

tmw.media API Active

The tmw.media API is a fast, RESTful JSON web API used by the internet radio, GGRadio It offers access to our now playing, recently played and
currently displayed artwork in a simple to use JSON format. While primarily used by tmw.media's own websites, it can be used by anyone
and new features are being added all the time, such as a WebSocket, song-request feature and song-runtime information
It is written in Node.JS and uses Docker for containerization.

Mixer Retweet / Retweet X Active

MixerRetweet (MXRRT, RTX) is an advanced Twitter auto-retweet bot used primarily by Mixer Retweet.
An open-source version of this project will be available soon, but now for it remains close-sourced until the API is fully developed, and the bugs are ironed out.

Virtual Market Discord Bot Active

VM-Bot is another project I developed for tmw.media - it's a fairly simple Discord bot for Virtual Market (a virtual goods trading forum)
It has moderation, rule searching, automatic role management, anti-spam features and welcome messages.
Once the codebase has been tidied up, I will make this project open source.

HeroPro In Development

'HeroPro' is an in-development digital signage system built in TypeScript and distributed using Docker Containerization
It not only has basic web-based viewers, but also makes use of Windows' SCR files, to allow dormant computers
to be used for advertising/announcements.

Show Text-In Platform Retired

During the George Greenaway 24 and 36 hour radio marathons, I wrote a text-in system that collects SMS messages, Facebook messages and Tweets to
display them on a simple web interface. It was written in Node.JS, and used basic HTML/CSS/JS for the frontend.
This project has been retired for the time being, it will remain in this state unless it's needed again in the future
I have decided not to make this project open source, mostly because it was written in a hurry, so the code base isn't... perfect.
